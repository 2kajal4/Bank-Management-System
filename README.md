# Bank Management System

## Features
- Admin & User roles
- Account creation
- Deposit & Withdraw
- Transfer money
- Transaction history

## Tech Used
- Core Java
- OOP Concepts
- File Handling

## Project Structure
- app → UI
- service → Logic
- model → Data classes
- util → File handling

## How to Run
1. Run BankApp.java
2. Select Admin/User
3. Perform operations
